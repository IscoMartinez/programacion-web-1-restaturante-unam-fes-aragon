# programacion-web-1-restaturante-unam-fes-aragon
## La Cocinita – Sitio Web

### Proyecto de Programación Web I

Descripción

Este proyecto es un sitio web estático para el restaurante La Cocinita, diseñado con HTML y CSS.
Incluye páginas con información del restaurante, menú, promociones, galería de imágenes y recetas.

El objetivo es practicar la estructura de un sitio web, el uso adecuado de etiquetas HTML, estilos CSS externos y organización de archivos.
